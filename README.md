❄️ FrostManut – Sistema de Manutenção de Ar-Condicionado
📋 Sobre o Projeto

O FrostManut é uma plataforma desenvolvida para otimizar o gerenciamento de serviços de manutenção de ar-condicionado. O sistema permite controlar clientes, técnicos, equipamentos, ordens de serviço e agendamentos em um único ambiente, proporcionando mais organização, produtividade e eficiência operacional.

🎯 Objetivo

Facilitar o processo de gestão de empresas de manutenção de climatização através de uma solução moderna, intuitiva e centralizada.
